About

With a strong foundation in both traditional sound design and software engineering, I specialize in creating dynamic audio systems for interactive media. My goal is to reduce manual labor through automation and empower creative teams with robust tools.

Whether it's scripting Python tools for batch processing, setting up complex Blueprints in Unreal Engine, or profiling memory in Wwise, I thrive in the intersection of art and logic.

Pre-production & Naming Conventions

Establishing a rock-solid foundation is crucial. I design macro-structures for audio events and define strict naming conventions to ensure WAAPI and automation scripts can parse data flawlessly later in the pipeline.

Tool Building & Automation

I write custom Python scripts and Reaper scripts (ReaScript) to batch export files, generate metadata, and automatically import them into Wwise using the Wwise Authoring API (WAAPI), saving hours of manual labor.

client.call("ak.wwise.core.object.create", ...)

Implementation & Profiling

Hooking up the audio in Unreal Engine via Blueprints or C++. I set up RTPCs, State Groups, and occlusion/obstruction systems. Finally, I profile the game to ensure memory limits and CPU usage are within budget.
